# 📸 Rakshak AI - Project Screenshots & Visual Documentation

This directory contains key visual assets and UI screenshots showcasing the Rakshak AI Crisis Command Platform:

- **Command Center Dashboard (`Command_center_view_1.png`):** Real-time monitoring grid with threat gauges, facility views, and dispatch feeds.
- **Threat Detection (`Detection.png`):** Ingestion of IoT sensor signals and crisis anomaly recognition.
- **Detection & Reasoning Analysis (`Detection_Analysis.png`):** Transparent LangGraph AI reasoning panel exposing threat analysis.
- **Final Emergency Approval (`Final_aproval.png`):** Human-in-the-loop confirmation modal for autonomous dispatches.
- **Neural Engine Core (`Neural_engine.png`):** Central pulsing visual indicator tracking AI status and system threat severity.
- **Alerting & Route Navigation (`alerting and route.png`):** Dynamic OSRM/Overpass routing from nearest emergency stations to the crisis location.
- **Authentication & Login (`login.png`):** Streamlined crisis commander sign-in interface.
- **Command Center Selection (`selection of command center.png`):** Facility and venue selection interface.
